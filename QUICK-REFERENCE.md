## ⚡ Quick Prompts

### Task Product Agent for Personas
```
You are the Product Agent for AceQuest, an educational assessment
platform for grades 3-8.

TASK: Create detailed user personas for:
1. Students (grades 3-5 and 6-8)
2. Parents
3. Teachers

Include demographics, goals, pain points, and what success looks like.
```

### Task Engineering Agent for Tech Stack
```
You are the Engineering Agent for AceQuest.

TASK: Recommend technology stack for:
- Frontend (React/Next.js?)
- Backend (Node.js/Python?)
- Database (PostgreSQL?)
- Hosting (AWS/Vercel?)

Timeline: 3 months to MVP
Scale: 10,000+ concurrent users eventually
Budget: $[X]/month for infrastructure

Provide recommendations with rationale.
```

### Task Marketing Agent for GTM
```
You are the Marketing Agent for AceQuest.

TASK: Develop go-to-market strategy

Goals:
- 500 students in 3 months
- 5 school partnerships
- 2,000 email subscribers

Budget: $[X]/month
Target: Parents of students grades 3-8

Deliverables:
1. Channel strategy
2. Month-by-month marketing roadmap
3. Content strategy
4. Partnership approach
```

## 📋 Weekly Rhythm

### Monday (Planning)
- Review last week's progress
- Set this week's priorities
- Task agents with weekly goals

### Daily (5-10 min)
- Check agent progress
- Answer agent questions
- Unblock issues

### Wednesday (Mid-week)
- Review work-in-progress
- Adjust priorities if needed

### Friday (Review)
- Review completed work
- Approve or request revisions
- Plan next week

## 🚨 Red Flags & Fixes

| Problem | Solution |
|---------|----------|
| **Agents need constant clarification** | Use more specific prompts with examples |
| **Low quality deliverables** | Provide examples, raise standards, iterate |
| **Work not aligned across agents** | Better coordination, document decisions |
| **You're doing all the work** | Trust agents more, delegate fully |
| **Progress too slow** | Focus on fewer priorities, increase urgency |

## 🎨 Brand Guidelines

### Voice & Tone
- **Adventurous** but not childish
- **Supportive** but not condescending
- **Achievement-oriented** but not stressful
- **Fun** but focused on learning

### Key Messages
**For students:** "Turn every assessment into an adventure"
**For parents:** "Track progress, support growth"
**For teachers:** "Data-driven insights, student engagement"

### Visual Style
- **Colors:** Bright, energetic (not distracting)
- **Typography:** Clear, readable, age-appropriate
- **Imagery:** Diverse students, achievement, adventure themes
- **Icons:** Simple, friendly, consistent
