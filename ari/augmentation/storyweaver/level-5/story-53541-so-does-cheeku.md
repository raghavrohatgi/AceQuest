# So Does Cheeku

The sun gets up. So does Cheeku.

Kittens eat. So does Cheeku.

Puppies play. So does Cheeku.

Birds have a bath. So does Cheeku.

Peacocks dance. So does Cheeku.

The sun goes to sleep. So does Cheeku.

The moon gets up and smiles. In his dream, so does Cheeku.
