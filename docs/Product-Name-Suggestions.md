## 💰 DOMAIN ACQUISITION TIPS

### If .com is Taken
**Priority order:**
1. **[name].io** - Tech-friendly, modern (e.g., acequest.io)
2. **[name].app** - Perfect for app-first products (e.g., questlearn.app)
3. **get[name].com** - Common workaround (e.g., getacequest.com)
4. **try[name].com** - Alternative workaround (e.g., trylearnlift.com)

### Price Expectations
- **New registration:** $10-15/year (.com), $25-40/year (.io), $15-20/year (.app)
- **Premium/taken domains:** $500-$10,000+ (negotiate if you really want it)
- **Recommendation:** Don't overspend on domain for MVP - pick an available alternative

### Domain Registrars
- **Namecheap** - Good prices, easy to use
- **Google Domains** (now Squarespace) - Clean interface
- **Cloudflare** - Best prices, minimal markup
- **Avoid:** GoDaddy (aggressive upselling)

## 📝 TRADEMARK CONSIDERATIONS

Once you choose a name:

### Before Committing
1. Search USPTO database (uspto.gov/trademarks)
2. Check for "education" or "computer software" categories (Class 9, 41, 42)
3. Look for "confusingly similar" names

### After Launch
1. File for trademark ($250-350 per class)
2. Consider hiring trademark attorney for comprehensive search ($500-1000)
3. Monitor for infringement

### Quick Check
Search: [your name] + "trademark" or "TM" on Google
Check: https://www.uspto.gov/trademarks/search
