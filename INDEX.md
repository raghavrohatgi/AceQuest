### Task Management

#### tasks/task-template.md
**Purpose:** Standardized template for tracking agent tasks
**When to use:** For every significant task you assign
**What's inside:**
- Task metadata (status, priority, due date)
- Context and goal
- Requirements and deliverables
- Dependencies
- Agent prompt (copy/paste what you used)
- Progress log
- Review and feedback section
- Testing/validation
- Next steps
- Lessons learned

**How to use:**
1. Copy template for each new task
2. Fill in the details before tasking agent
3. Paste the prompt you actually used
4. Track progress and decisions
5. Document lessons learned
